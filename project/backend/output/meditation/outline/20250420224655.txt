# 冥想配置

- 类型：affirmation
- 时长：5分钟
- 音乐风格：calm

# 用户意图

- 目标：增强面试前的信心
- 当前情绪：紧张
- 特殊偏好：audio_format: 冥想音频

# 内容结构

## 内容块 1
目的：引导放松和进入冥想状态
关键要素：深呼吸, 放松身体, 专注当下
语气：温和
时间：1分钟

确定内容：
- 找一个安静舒适的地方坐下或躺下，轻轻闭上眼睛。
  上下文：开始冥想
- 深吸一口气，慢慢呼出，感受身体的放松。
  上下文：放松身体

放置位置：start

## 内容块 2
目的：重复肯定语以增强信心
关键要素：肯定语, 自信, 专注
语气：坚定
时间：3分钟

确定内容：
- 我充满自信地迎接下周的面试，展现出最好的自己。
  上下文：肯定语
- 我准备充分，轻松应对面试中的每一个问题。
  上下文：肯定语
- 我的能力和经验让我在面试中脱颖而出。
  上下文：肯定语
- 我保持冷静和专注，展现出我的专业素养。
  上下文：肯定语
- 面试官对我的表现印象深刻，我顺利通过了面试。
  上下文：肯定语

放置位置：middle

## 内容块 3
目的：结束冥想并回归现实
关键要素：感恩, 回归, 目标
语气：温暖
时间：1分钟

确定内容：
- 感谢自己花时间专注于提升信心。
  上下文：感恩
- 慢慢睁开眼睛，带着这份自信回到当下。
  上下文：结束冥想

放置位置：end

# 特殊指示

- 背景音乐风格为轻柔的钢琴或自然声音
- 语速适中，语气坚定而温和
